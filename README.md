# Model maker

## Info
https://www.tensorflow.org/lite/models/modify/model_maker/image_classification

## Prerequisites / known good configuration
* Windows 11
* Python 3.9.13
* pip 24.0
* pip-tools 7.4.1

## Prepare environment
```
pip install pip-tools
pip-compile
pip install --upgrade -r requirements.txt
```

## Run
```
python main.py
```
